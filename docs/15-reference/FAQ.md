## Getting Started

**Q: How do I get started?**
A: Follow the [Quick Start Guide](../01-getting-started/QUICK-START.md) - takes 5 minutes!

**Q: What do I need to run this?**
A: Node.js 20+, a Supabase account, and optionally Claude API key. See [Prerequisites](../01-getting-started/PREREQUISITES.md).

## Development

**Q: Why is the dev server on port 3014?**
A: To avoid conflicts with other versions (V3, V4, V11, V12, etc.) during development.

**Q: Do I need all the integrations?**
A: No! Only Supabase is required. Claude, Zoho, Jira are optional and add specific functionality.

## Deployment

**Q: How do I deploy to production?**
A: We support Docker and Vercel. See [Deployment Guide](../10-deployment/DEPLOYMENT-GUIDE.md) or `/Aldo/PRODUCTION-DEPLOYMENT.md`.
